luci-YAWAC
==========
This package contains script Yet Another Wifi Auto Connect for LuCI

How to install
--------------
1. Connecting router with the internet.

2. Doing opkg update first.
   ```shell
   opkg update
   ```

3. Installing package.
   ```shell
   opkg install
   ```

4. Reboot router.
   ```shell
   reboot
   ```

7. Finally rebooting your router.
   ```shell
   reboot
   ```
